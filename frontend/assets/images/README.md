# Logo Directory

## Hướng dẫn thêm logo

1. Đặt file logo của bạn vào thư mục này với tên file: `logo.png`
2. Logo nên có:
   - Định dạng: PNG (có nền trong suốt)
   - Kích thước đề xuất: Tối thiểu 200x200px (sẽ được tự động scale)
   - Màu sắc: Phù hợp với theme (nên có màu hồng theo thiết kế)

3. File logo sẽ được hiển thị trong navbar của tất cả các trang
4. Logo có thể click để trở về trang chủ (index.html)

## Kích thước hiển thị
- Desktop: 50px chiều cao
- Tablet (max-width 768px): 40px chiều cao
- Mobile (max-width 480px): 35px chiều cao
